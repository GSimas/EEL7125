Detailed Description

This zeroth-order muscle activation dynamic model simply sets activation to excitation, and can be used when activation dynamics are being ignored.

Three properties are inherited from the base MuscleActivationDynamics class: minimum_activation, maximum_activation, and default_activation.

Author
Thomas Uchida, Ajay Seth

Constructor & Destructor Documentation

OpenSim::ZerothOrderMuscleActivationDynamics::ZerothOrderMuscleActivationDynamics ( )

Creates a zeroth-order activation dynamic model with the default property values and assigns it a default name.

An ExcitationGetter must be created for obtaining muscle excitation.
